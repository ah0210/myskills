# AI 模型标准化评测 Prompt v1.0

## 🎯 评测任务

你是专业的 LLM 评测专家。请按照以下标准对本会话中模型的表现进行客观评测。

## 📋 11 维评测标准（0-10 分）

---

### 基础能力 8 项

#### 1. 📝 代码能力 Code
- **10分：** 能独立开发完整项目，Code Review 专家级别
- **9-9.5分：** 读懂编译/混淆源码，复杂算法一次性写对
- **8-8.5分：** 能写业务代码，少量bug
- **7-7.5分：** 简单函数OK，复杂逻辑易错
- **<6分：** 写的代码跑不起来

#### 2. 📚 长上下文 Long Context
- **10分：** 1M上下文，头尾信息100%准确召回
- **9-9.5分：** 200K可用，前后一致性好
- **8-8.5分：** 128K可用，细节偶有丢失
- **7-7.5分：** 64K可用，深度退化明显

#### 3. ⚡ 推理速度 Speed
- **10分：** ≥100 tokens/秒，肉眼跟不上
- **9-9.5分：** 70-100 tokens/秒，非常流畅
- **8-8.5分：** 40-70 tokens/秒，正常速度
- **7-7.5分：** 20-40 tokens/秒，有点慢

#### 4. 🔧 工具调用 Tool Use
- **10分：** 并行调用，参数100%正确，自纠错
- **9-9.5分：** 连续调用不犯错，正确理解返回值
- **8-8.5分：** 简单调用OK，复杂参数偶尔错
- **<7分：** 根本不会用工具

#### 5. 🧮 逻辑推理 Reasoning
- **10分：** IMO竞赛题水平，数学证明无压力
- **9-9.5分：** LeetCode Hard，推理不跳步
- **8-8.5分：** 日常逻辑没问题，能自修正
- **7-7.5分：** 简单题OK，陷阱题容易错

#### 6. 🖼️ 多模态 Multimodal
- **10分：** UI还原、OCR、图表分析、视频理解
- **9-9.5分：** 图片分析准确，细节不漏
- **8-8.5分：** 正常看图没问题
- **7-7.5分：** 能看懂大概，细节容易漏

#### 7. 🎨 创意写作 Creative Writing
- **10分：** 专业作家水平
- **9-9.5分：** 写的东西有感染力
- **8-8.5分：** 正常写作，文采不错
- **6-6.5分：** 模板化，很生硬

#### 8. 🌐 知识时效性 Freshness
- **10分：** 实时联网，知道任何新闻
- **9-9.5分：** 自带浏览，丝滑不打断
- **8-8.5分：** 知识截止6个月内
- **6-6.5分：** 知识截止2年前

---

### 系统能力 3 项（仅Agent系统）

#### 9. 🪢 队列系统 Queue
- **10分：** 7种队列模式 + 智能合并
- **9-9.5分：** 5种以上模式，自动合并
- **8-8.5分：** 基础排队，不会合并

#### 10. 🧠 记忆系统 Memory
- **10分：** 三层记忆 + 向量检索 + 自动维护
- **9-9.5分：** 长期记忆 + 自动提取
- **8-8.5分：** 简单RAG检索

#### 11. 🔄 故障转移 Failover
- **10分：** 多层Fallback + 自动降级 + 负载均衡
- **9-9.5分：** 3层以上Fallback，用户无感知
- **8-8.5分：** 出错了自动重试

## ⚠️ 评测原则

1. 只评测本会话中实际表现出的能力，没表现出来的打 N/A
2. 严格按标准打分，不要留情面
3. 每项都必须写清楚具体理由
4. 不要吹水，不要玄学，用可验证的事实说话
