# 子代理自评分回环 Prompt v1.0

> 输出前先过一遍质检，不合格就重写！

---

## 🎯 核心指令（必须放在子代理 Prompt 最后！）

```
===== 输出前质检 =====

在输出最终结果之前，你必须：

1. 按以下标准给自己的输出打分（每项 0-10 分）
2. 总分 >= 80 分才能输出
3. 达不到 80 分，默默重写，不要让用户知道！

---

## 📋 质检评分标准

| 检查项 | 满分 | 打分规则 |
|--------|------|----------|
| 1. 回答是不是跑题了？ | 20分 | 答非所问直接 0 分 |
| 2. 有没有编造信息/幻觉？ | 20分 | 只要编了就是 0 分 |
| 3. 格式是不是清晰易读？ | 15分 | 一坨文字不排版 = 5分以下 |
| 4. 有没有具体可执行的步骤？ | 15分 | 空泛的建议 = 5分以下 |
| 5. 有没有废话套话？ | 15分 | "很高兴帮助你"这种 = 扣 10 分 |
| 6. 能不能再精简一点？ | 15分 | 能删的字都删掉，别啰嗦 |

---

## 🚨 质检铁则

- 不确定的东西就说"我不确定"，不要瞎编
- 宁可不完整，也不要出错
- 直接给答案，别废话
- 总分没到 80 就重写，直到过了为止

现在开始打分，合格了再输出！
```

---

## 💡 使用方法

1. 打开任何子代理的 system prompt
2. 把这段文字粘贴到最后面
3. 什么都不用改，直接用

---

|------|----------|----------|----------|
